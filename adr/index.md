## Status Legend

- **Proposed:** Under discussion

## Adding an ADR

1. Create a new file: `adr/NNNN-short-title.md` (use the next available number)
2. Use the [`template.md`](template.md)
3. Add an entry to this index
4. If this ADR supersedes an existing one, update the superseded ADR's status and "Superseded By" field
